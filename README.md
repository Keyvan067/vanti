# VANTI - WooCommerce Theme

A modular, OOP-based WooCommerce theme with Elementor integration, built with PSR-4 autoloading and modern PHP practices.

## Features

- 🏗️ **Modular Architecture** - Organized code structure with separate modules for different functionality
- 🎯 **OOP Design** - Object-oriented programming for maintainable and scalable code
- 🔄 **PSR-4 Autoloading** - Automatic class loading with Composer
- 🛒 **WooCommerce Ready** - Full WooCommerce support and customization capabilities
- 🎨 **Elementor Compatible** - Ready for Elementor page builder integration
- 📱 **Responsive Design** - Mobile-first approach
- ⚡ **Performance Optimized** - Clean and efficient code

## Installation

1. Download or clone into WordPress themes directory
2. Run `composer install` (optional)
3. Activate theme in WordPress
4. Install WooCommerce plugin

## Quick Start

The theme is ready to use out of the box with modular architecture.
